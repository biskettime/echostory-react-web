# EchoStory - Interactive Story Platform

EchoStory는 사용자가 AI 캐릭터와 상호작용하며 스토리를 즐길 수 있는 인터랙티브 스토리 플랫폼입니다.

## 🌟 주요 기능

### 📚 스토리 시스템
- **인터랙티브 스토리**: AI 캐릭터와 실시간 대화
- **스토리 생성**: 사용자가 직접 스토리와 캐릭터 생성 가능
- **다양한 장르**: 로맨스, 판타지, 일상, 미스터리 등
- **이미지 갤러리**: 스토리별 이미지 컬렉션

### 👥 소셜 기능
- **크리에이터 팔로우**: 좋아하는 작가 팔로우
- **활동 추적**: 대화, 앨범, 즐겨찾기, 좋아요 관리
- **크리에이터 프로필**: 작가 정보 및 작품 목록

### 🎨 사용자 경험
- **모바일 최적화**: 반응형 디자인
- **다크 테마**: 눈에 편한 어두운 테마
- **직관적 네비게이션**: 하단 탭 네비게이션
- **검색 기능**: 스토리 및 크리에이터 검색

## 🛠 기술 스택

- **Frontend**: React 18, TypeScript
- **Styling**: Tailwind CSS
- **Build Tool**: Vite
- **Icons**: Lucide React
- **State Management**: React Hooks

## 📱 화면 구성

1. **홈**: 인기/신작 스토리 탐색
2. **생성**: 스토리 작성 및 관리
3. **활동**: 대화 기록, 앨범, 팔로우 관리
4. **프로필**: 사용자 설정 및 정보

## 🚀 시작하기

### 설치
```bash

```

### 개발 서버 실행

```bash

```

## 📂 프로젝트 구조

```
src/
├── components/          # React 컴포넌트
│   ├── ui/             # 재사용 가능한 UI 컴포넌트
│   └── figma/          # Figma 디자인 컴포넌트
├── data/               # 데이터 관리
│   ├── stories/        # 스토리 데이터베이스
│   └── chatSessions/   # 채팅 세션 관리
├── imports/            # 에셋 임포트
└── public/
    └── images/         # 이미지 리소스
```

## 🎯 주요 컴포넌트

- **HomeScreen**: 메인 화면, 스토리 탐색
- **ChatScreen**: AI 캐릭터와의 대화 인터페이스
- **StoryDetailScreen**: 스토리 상세 정보 및 갤러리
- **ActivityScreen**: 사용자 활동 관리
- **CreatorProfileModal**: 크리에이터 프로필 정보

## 🔧 데이터 관리

- **스토리 데이터베이스**: 메모리 기반 스토리 저장
- **채팅 세션**: 로컬 스토리지 기반 대화 기록
- **사용자 설정**: 로컬 스토리지 기반 설정 저장

## 🎨 디자인 시스템

- **컬러 팔레트**: 다크 테마 기반
- **타이포그래피**: Inter, Noto Sans KR
- **컴포넌트**: 일관된 디자인 시스템
